# Запуск проекта в VS Code

## Что нужно установить

Перед запуском нужны только:

- JDK 17;
- VS Code.

Maven отдельно устанавливать не обязательно: в backend добавлен Maven Wrapper.
Node.js для обычного запуска не нужен.

## Как запустить

1. Скачать проект с GitHub.
2. Открыть папку проекта в VS Code.
3. Нажать `Ctrl+Shift+B`.
4. Выбрать задачу `Start app`, если VS Code спросит.
5. Открыть в браузере:

```text
http://localhost:8080/
```

REST API доступен на:

```text
http://localhost:8080/api/tasks
```

## Демо-данные

Когда приложение уже запущено, можно заполнить список демонстрационными задачами:

1. Открыть `Terminal -> Run Task...`.
2. Выбрать `Seed demo data`.
3. Обновить страницу `http://localhost:8080/`.

Для этой дополнительной команды нужен Node.js.

## Остановка

Остановить приложение можно в терминалах VS Code сочетанием `Ctrl+C`.
